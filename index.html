📋 About This Project

This is a remote session manager for Chromium. It allows you to:

  • ✅ Control Chromium browser instances remotely
  • ✅ Run automated browser tasks and navigation
  • ✅ Execute JavaScript in browser context
  • ✅ Capture screenshots
  • ✅ Manage multiple sessions simultaneously

🚀 API Endpoints

  • POST /api/sessions/start - Start a new Chromium session
  • POST /api/sessions/:id/navigate - Navigate to URL
  • POST /api/sessions/:id/execute - Execute JavaScript
  • GET /api/sessions/:id/screenshot - Take screenshot
  • POST /api/sessions/:id/close - Close session
  • GET /api/status - Get server status
