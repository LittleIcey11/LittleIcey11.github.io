🚀 Deploy to Railway

This will deploy your Chromium server. Once deployed, you'll get a URL to use.

📋 What This Does

  • ✅ Runs Chromium in the cloud
  • ✅ Provides API endpoints for browser control
  • ✅ Start/stop sessions remotely
  • ✅ Navigate, screenshot, execute JavaScript

🔗 API Endpoints

  • POST /api/sessions/start
  • POST /api/sessions/:id/navigate
  • GET /api/sessions/:id/screenshot
  • POST /api/sessions/:id/execute
  • POST /api/sessions/:id/close
  • GET /api/status

📖 Local Development

npm install
npm start

Then visit: http://localhost:3000

💡 After Deployment

Update your code with the Railway URL and redeploy to connect the frontend to your server.
