🌐 Chromium Remote Sessions

Run Chromium browser sessions locally with a simple web interface.

Checking Chromium status...

📋 How to Use

This app requires the Node.js server to be running locally on your machine.

npm install
npm start

Then visit: http://localhost:3000

✨ Features

  • ✅ Start and manage Chromium browser sessions
  • ✅ Navigate to URLs programmatically
  • ✅ Execute JavaScript in browser context
  • ✅ Capture screenshots
  • ✅ Multiple independent sessions

🚀 API Endpoints

  • POST /api/sessions/start - Start a new session
  • POST /api/sessions/:id/navigate - Navigate to URL
  • POST /api/sessions/:id/execute - Execute JavaScript
  • GET /api/sessions/:id/screenshot - Take screenshot
  • POST /api/sessions/:id/close - Close session
  • GET /api/status - Get server status
